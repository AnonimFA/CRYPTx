Installing for Linux/Termux:
$ apt update
$ git clone https://github.com/AnonimFA/CRYPTx
$ cd CRYPTx
$ apt install python3
$ pip install requests
$ pip install pyAesCrypt
$ pip install subprocess
$ python3 cryptx.py

Installing for Windows Command Line Interpreter:
$ git clone https://github.com/AnonimFA/CRYPTx
$ cd CRYPTx
$ pip install requests
$ pip install pyAesCrypt
$ pip install subprocess
$ python3 cryptx.py
