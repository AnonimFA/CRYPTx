# CRYPTx
Beta version 1.0.0 by AnonimFA.
• CRYPTx - Files crypter. Now it have only BASE64 crypting.
• Tested with Kali Linux and Termux.
• This script can work with big files.

Installing:
$ apt update
$ apt install python3 
$ pip install requests
$ pip install subprocess
$ python3 cryptx.py
